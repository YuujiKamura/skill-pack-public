# 下請け業者宛てメール作成（見積依頼・施工体制資料依頼）

## メール文面の基本構成

1. **宛名**（会社名＋担当者名）
2. **挨拶**（「お世話になっております」or「平素よりお世話になっております」）
3. **工事名・施工時期**
4. **依頼内容**（見積、材料確認資料、施工体制資料など）
5. **数量詳細**（必要に応じて）
6. **添付ファイル案内**
7. **結び**（「お忙しいところ恐縮ですが、よろしくお願いします/申し上げます」）
8. **署名**

---

## 文面作成のルール

| 項目 | NG | OK |
|------|----|----|
| 工事名の〇マーク | 〇市道〜 | 市道〜 |
| 見出しの使用 | 【お願い事項】【参考資料】 | 必要最小限に |
| 施工時期 | 記載なし | 「2026年2月初旬に施工予定」 |
| 夜間施工 | 別行で記載 | 工事名または数量に含める |
| スプレッドシートリンク | 本文に記載 | 添付で十分（削除） |
| 暫定数量の注釈 | 「記載がないため暫定数量」 | 「個別箇所の記載がなく、暫定」 |

---

## 挨拶・結びの使い分け

| 相手との関係 | 挨拶 | 結び |
|-------------|------|------|
| 頻繁にやり取り | お世話になっております | よろしくお願いします |
| やや丁寧に | 平素よりお世話になっております | よろしくお願い申し上げます |

## テンプレート例（区画線業者宛て）

```
[会社名]　[担当者名]様

お世話になっております、[自社名]の[担当者]です。
標記の件、見積書の作成と資料送付をお願いします。

[工事名]（夜間施工）
[施工時期]に施工予定

①見積書は以下の2パターンでお願いします
　・通常の溶融式
　・停止線と横断歩道を高輝度にした場合

②道路鋲を含め、材料確認資料の送付をお願いします

③施工体制資料（作業員名簿等）をメールで送付いただけると助かります

【区画線数量】※平面図に個別箇所の記載がなく、暫定です
＜白 W=15cm＞
・外側線: XXXm
・中央線: XXXm
（以下省略）

・平面図、設計書は添付ファイルをご確認ください

お忙しいところ恐縮ですが、よろしくお願いします。
```

## 依頼内容と宛先の組み合わせ

| 依頼内容 | 区画線業者 | 切削業者 |
|---------|-----------|---------|
| 見積依頼 | 営業担当 | 営業担当 |
| 材料確認資料 | 営業担当 | 不要（自社調達） |
| 施工体制資料 | 営業担当 | 営業担当 |
| 出来形精算 | - | 精算担当 |

※連絡先は別途管理

## 元請け・一次下請けの関係

### 注意点

- 「再下請負通知書」ではない（元請・一次下請の関係）
- 契約書は元請側で作成
- 下請けには「作業員名簿等」のみ依頼

---

## 添付ファイルの取得先

入札時フォルダから平面図・設計書を添付する。

---

## 区画線数量の取得元

施工計画書の「１内容」シートではなく、**材料確認願いスプレッドシート**から取得

### 数量記載方法

施工号（施1号〜施6号）ではなく、**幅と色で仕分け**

```
区画線工（溶融式）:

白 W=15cm:
  外側線、中央線、車線分離線...

白 W=30cm:
  交差点標示

白 W=45cm:
  停止線、横断歩道

黄 W=15cm:
  停車禁止標示

道路鋲:
  自発光式（円形）
```

## 高輝度区画線について

停止線・横断歩道には高輝度材料を使用するパターンの見積りも併せて依頼する

- 製品名: スーパーメガルクス
- 対象: 停止線、横断歩道
