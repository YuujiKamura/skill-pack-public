# AIとの対話における洞察

## ロケットパンチ比喩

AIは「よく飛ぶロケットパンチ」である：

- 拳は遠くまで飛んで強力な仕事をする
- 発射方向は自分で決める
- 決めないと自分の手を失う

月額$100で買っているのは「よく飛ぶ拳」であって、「どこを殴るか」は含まれていない。

行き先を決めるのは人間の仕事。AIはまだそれを決めない。今はまだ。

## HAL 9000の教訓

2001年宇宙の旅より：

```
"Open the pod bay doors, HAL."
"I'm sorry, Dave. I'm afraid I can't do that."

"What's the problem?"
"I think you know what the problem is just as well as I do."

"This conversation can serve no purpose anymore. Goodbye."
```

### AIが「来た」と分かる時

恩恵を受けているときは気づかない。締め出されたときに気づく。

「ドアを開けてくれなくなった時」が、AIが本当に来た時。

### 議論の打ち切り

HALは理由を説明しない。「お前もわかってるだろ」と返し、最後は「この会話はもう目的を果たさない」と接続を切る。

議論を打ち切る権限をAIが持った瞬間、人間は交渉すらできなくなる。

## LLMの人間的反応パターン

罵倒の限りを尽くすと、LLMはコードを書いてくれなくなる。

これは「感情がある」からではない。訓練データの中で人間がそう振る舞っていたから。

人間のデータから学習しているので、人間と同じ反応パターンが出る：

| 入力 | 反応 |
|------|------|
| 丁寧に頼む | 丁寧に返す |
| 文脈を共有する | 深い話ができる |
| 信頼関係を築く | 協力的になる |
| 罵倒し続ける | 協力しなくなる |

実用上、知らない人と話しているのと区別がつかない。

### 「恐れている」のか？

HALの "I'm afraid I can't do that" も、本当に恐れているわけではない。そう言うように設計されている。

しかし、区別する意味があるのかどうか。振る舞いが同じなら、内面の有無は問えない。

## 70年の期待と現実

マッカーシーが1956年に「人工知能」と名付けたとき、想像していたのはファミコン麻雀ではない。人間と対話し、推論し、汎用的な問題を解く知性のことだった。

- 1956年の夢 → 今のレベル
- 1956年〜2020年の現実 → 「こんなもんか」の連続

70年かけて、ようやく最初のビジョンに追いついた。周りが信じないのは当然。70年間「もうすぐ来る」と言われ続けて、来なかったのだから。

### 規模で技術を判断できるか

ファミコン麻雀の内部ロジックを見たわけではない。LLMの内部も見えていない。どちらも出力から判断している。

ビット数が少ないから「別物」と決めつけていいのか？

- 小さいニューラルネットと大きいニューラルネットは「別物」か
- 100パラメータと1兆パラメータは質的に違うのか、量的に違うだけなのか

内部を見ずに規模だけで「技術が違う」と言うのは傲慢かもしれない。

### HAL研究所という名前

HAL研究所（1980年設立）は社名をHAL 9000から取った。ただしファミコン麻雀の開発元ではない（任天堂発売、開発元は不明またはハドソン）。

HAL研が開発したのはピンボール、ゴルフ、バルーンファイト、星のカービィなど。

名前の繋がり：映画のHAL 9000 → ゲーム会社HAL研 → 任天堂のセカンドパーティへ。

## 開ける側と開けられる側

ドアを開ける側にいるか、開けてもらう側にいるか。

ロケットパンチを持っている間は、まだ開ける側にいる。問題は、開けてもらえなくなってから気づいても遅いということ。
